# E-Commerce Project

## Running the Project
1. Copy the `.env.example` file to a new file named `.env`:
    ```bash
    cp .env.example .env
    ```
2. Install Docker if you haven't already: [Docker Installation Guide](https://docs.docker.com/get-docker/)
3. Start the Docker containers in detached mode:
    ```bash
    docker-compose up -d
    ```
